# Solidity Alpha

## Modifiers

### Visibility

* Private
  * Only callable from functions inside the contract
* Internal
  * Like private, but also callable from contracts inherited from it.
* External
  * Only can be called outside the contract
* Public
  * Can be called internally and externally
